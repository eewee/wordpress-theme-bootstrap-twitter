= EEWEE BT =

* By the eewee team, http://www.eewee.fr/
* Tab & accordion system for your news.
* Page : home, full width, left sidebar, right sidebar, sitemap.
* Widget : ads (title, image, link. 4 ads possible).
* Menu : top menu, bottom menu.
* Theme options : general (logo, website size), carousel (active, desactive, title, description, image), networks, address, manual
* Lang : en, fr

== INSTALLATION ==

* Menu : apparence / menu => choose your "top menu" and "footer menu".
* Theme options / general : width image = 100px minimum.
* Theme options / network : This information appears in the block "about the author" in the foot of an article.
* Theme options / address : This information appears in the footer.
